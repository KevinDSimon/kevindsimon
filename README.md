 # README # 

## Installation Instructions ##

**Install Node.js**
http://nodejs.org/

**Clone repository**

**Install Bower and Grunt**

```
#!

npm install -g bower grunt-cli
```

**Install Node Packages**

```
#!

npm install
```

**Install Dependencies with Bower**

```
#!

bower install
```

**Other Dependencies include**

```
#!

gem install compass
```


## To Run ##

```
#!

grunt serve
```
